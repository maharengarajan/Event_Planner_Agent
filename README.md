# Event_Planner_Agent

1. Create virtual environmet
```
conda create -p venv python==3.10 -y
```
2. Activate virtual environment
```
conda activate venv/
```
3. Install req
```
pip install -r requirements.txt
```
4. Run app
```
python main.py
```
